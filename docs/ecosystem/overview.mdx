## Refresh cadence

Mirrored pages are reviewed at minimum **per release** of the Morpheus Lumerin Node. The `last_verified` frontmatter on each page tracks the last time its content was reconfirmed against the upstream source.

For live data — counts, prices, status — these mirrors **link out** rather than snapshot. A snapshot drifts; a link doesn't.
