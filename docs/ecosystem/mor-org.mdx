## What it is

[mor.org](https://mor.org) is the public-facing entry point for the Morpheus network. It's where new users land to understand what Morpheus is, who's behind it, and where to go next.

## What you can do there

- Read the project overview and pitch.
- Find links to the main user surfaces: [app.mor.org](https://app.mor.org) (hosted chat), [tech.mor.org](https://tech.mor.org) (calculators / tech docs), [active.mor.org](https://active.mor.org) (live network status), [myprovider.mor.org](https://myprovider.mor.org) (provider operator GUI), [apidocs.mor.org](https://apidocs.mor.org) (hosted [Morpheus Inference API](/inference-api/overview)).
- Find the white paper and economic model documentation.
- Find community channels (Discord, X, etc.).
